# Core Sampler Bias Analysis

Statistical investigation comparing **horizontal** vs **oblique** core samplers used to measure sugarcane quality at the mill intake.

## Background

Core samplers extract a representative sample from incoming sugarcane loads to measure juice quality variables — primarily sucrose content (`Pol%caña` in kg/t). The orientation of the sampler (horizontal vs oblique) may introduce a systematic measurement bias that affects how mills account for sugarcane quality and payments.

This study uses data from **Zafra 2023-24**, where the same cane loads were processed by three mills simultaneously:

| Mill | Sampler type |
|---|---|
| Santa Ana | Horizontal (under study) |
| La Unión | Oblique (reference) |
| Pantaleón | Oblique (reference) |

Since the same physical cane load passed through all three mills, any difference in readings can only be attributed to the sampler, not to cane quality variation.

## Analysis Phases

| Phase | Script | Description |
|---|---|---|
| 1 | `phase1_data_prep.py` | Load, validate pairing, build paired dataset |
| 2 | `phase2_descriptive.py` | Summary statistics, normality tests, box plots |
| 3 | `phase3_paired_tests.py` | Paired t-tests, effect sizes |
| 4 | `phase4_method_comparison.py` | Bland-Altman, Passing-Bablok regression, CCC |
| 5 | `phase5_anova.py` | Blocked ANOVA, Tukey HSD, cut type interaction |
| 6 | `phase6_temporal.py` | Weekly bias trends, seasonal stability |
| 7 | `phase7_economic.py` | Economic impact of the bias in Pol%caña |

## Setup

```bash
pip install -r requirements.txt
```

Place the raw data file at `data/raw/Base de datos.csv`, then run:

```bash
python main.py
```

Or run phases individually from the `src/` folder:

```bash
cd src
python phase1_data_prep.py
```

## Key Variables

| Variable | Description |
|---|---|
| `%jugo` | Juice extraction percentage |
| `Brix JE (%)` | Brix of extracted juice |
| `Pol JE (%)` | Sucrose content of extracted juice |
| `Pza JE` | Purity of extracted juice |
| `Fibra%Caña` | Fiber content (% of cane) |
| `Pol%caña (kg/t)` | Sucrose per ton of cane — primary variable |
